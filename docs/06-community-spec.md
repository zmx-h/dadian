# 社区 (Community) — Specification

> 版本: v1.6 | 日期: 2026-06-21 | 原 product 中缺失此板块的详细设计

---

## 1. Overview

「社区」是搭电社交闭环的收口——用户在这里浏览同频人的出行故事，发现被命运眷顾过的路线和盲盒，查看城市玩家年鉴，从他人的分享中获得下一次出发的灵感。社区不是广场、不是论坛，而是一个安静的「同频客厅」——用户因为出行风格的共振而被连接，不因为热度而被排名。

## 2. Design Principles

| 原则 | 说明 |
|------|------|
| **同频优先于热度** | 算法匹配风格相近的人，而非推荐最热门的内容 |
| **低表达压力** | 留言限 100 字，不支持长文/嵌套回复，像路过点头而非辩论 |
| **发现驱动** | 核心价值是「从别人的出行中获得自己的灵感」 |
| **隐私可控** | 回忆录默认私密，用户主动选择分享到社区 |
| **去中心化** | 没有 KOL、没有粉丝数排名、没有热度榜 |

## 4. User Flows

### 4.1 浏览记忆长河

- 默认展示按「同频匹配度」排序的公开回忆录对卡片流
- 每张卡片展示：出行标题、用户头像+昵称、叙事风格标签、3 张缩略图、充电数、出行日期
- 卡片采用毛玻璃底 + 悬浮微光，hover 时展开更多预览
- 点击卡片进入回忆录详情（共用 回忆 板块的影卷组件，但只读模式）
- 瀑布流/网格布局，非列表布局——每条回忆都有视觉重量

### 4.2 同频匹配

系统根据以下维度计算用户间的风格相似度：
- 社交电量偏好区间
- 常选职业角色
- 出行时间模式（周末 vs 工作日，下午 vs 晚上）
- POI 品类偏好（咖啡馆派 vs 餐厅派 vs 户外派）
- 命运骰子接受率

匹配结果仅用于内容推荐和同频人发现，不显示具体分数，不排榜。

### 4.3 收藏出行剧本

- 在他人回忆录详情页，底部有 [📋 收藏此剧本] 按钮
- 收藏后，该出行路线进入自己出发页的「灵感路线」面板
- 用户可以直接用他人的路线作为模板，替换自己的起终点和电量偏好，生成自己的出行

### 4.4 足迹留言

- 在他人回忆录下方，显示现有留言（最多展示 50 条）
- 留言输入区：单行输入框 + [发送] 按钮，前端限制 100 字
- 不支持回复、不支持嵌套、不支持表情包
- 可给现有留言「充电」（点亮），互相充电的两人解锁「共振」标记
- 留言排序：按充电数（默认），也可切换为时间序

### 4.5 私信

- 入口：用户头像 → [发私信]
- 解锁条件：双方互关 + 至少一次相似出行记录（同区域打卡 / 同电量模式 / 同职业）
- 未解锁时显示：「你们还没有走过相似的路，暂时不能发私信哦」
- 解锁后显示共同出行标签：「你们都去过灯塔书屋」「你们都是特工」
- 私信功能轻量化：不支持图片/表情/语音，纯文本

### 4.6 城市玩家年鉴

- 入口：社区页右侧面板 (Desktop) / 顶部横向滚动 (Tablet)
- 年鉴是非竞技榜单，按角色维度分类：

| 榜单 | 判定逻辑 | 炫耀点 |
|------|---------|--------|
| 特工之王 | 打卡拍照数 + VLM 评价数 | 「拍了 342 张赛博大片」 |
| 美食雷达 | 美食品鉴官出行次数 + 菜品鉴定数 | 「发现了 27 家隐藏小店」 |
| 佛系大师 | NPC 模式出行次数 + 最低电量出行 | 「用 10% 电量出了 18 次门」 |
| 逃跑大师 | 逃离骰子次数（荣誉反向指标） | 「逃了 42 次，命运拿我没办法」 |
| 王家卫传人 | 王家卫风格回忆录数量 | 「生在王家卫的电影里」 |

- 每个榜单展示 Top 50，用户可查看自己排名
- 可选择隐藏自己的某个榜单排名
- 榜单入口风格：木制风格铭牌，非竞技体育排行榜

### 4.7 同频人发现

- 右侧面板展示 5 个同频人迷你卡片
- 每张卡片：头像、昵称、相似标签（如「电量低」「咖啡党」）、最近出行时间
- 点击进入该用户主页：公开回忆录 + 公开成就
- 「换一批」按钮重新随机推荐

## 5. UI Requirements

- Feed 卡片：毛玻璃底，2:3 比例封面图预览区 + 底部信息栏（头像+标题+统计）
- 卡片 hover：边框琥珀色微光 + scale(1.02) + 模糊照片变清晰
- 留言区域：半透明暗底 + 左侧装饰线 + 100 字计数
- 充电按钮：⚡ 图标，点亮后琥珀色呼吸
- 年鉴入口：木纹/铜色风格的铭牌卡片，与整体赛博浪漫风格区分（暗示「年鉴」的传统感）
- 同频人卡片：小尺寸圆形头像 + 一行标签 + 相似度光环（百分比不显示数字，用光环粗细暗示）
- 收藏剧本按钮：毛玻璃胶囊，点击后琥珀脉冲确认
- 所有动画使用 @vueuse/motion spring 或 CSS @keyframes，物理曲线优先
- 所有文案 Space Grotesk（标题）+ DM Sans（正文），数字 IBM Plex Mono

## 6. Moderation & Safety

- 公开回忆录先发后审：发布后即时可见，异步送审阿里云内容安全 API
- 留言实时敏感词过滤
- 用户可举报回忆录/留言（理由选择：不当内容/广告/隐私侵犯）
- 被举报内容临时隐藏，审核后决定恢复或移除
- 用户可屏蔽另一个用户（互不可见）

## 7. Community Health Metrics

| 指标 | 目标 | 说明 |
|------|------|------|
| 公开分享率 | > 30% | 用户完成出行后选择公开分享的比例 |
| 留言充电率 | > 15% | 留言被至少 1 次充电的比例 |
| 回报互动率 | > 8% | 收到充电后回充的比例（共振标记） |
| 举报率 | < 0.5% | 被举报内容占总内容的千分比 |
| 收藏转化率 | > 10% | 收藏剧本后实际出行的比例 |
